# MTK Property Management

A comprehensive property management module for Altensor ERP, handling apartments, residents, billing, payments, maintenance requests, and expenses.

## Features

- Building and apartment management
- Resident tracking
- Monthly billing system
- Payment tracking
- Maintenance request management
- Expense tracking
- Dashboard with KPI cards
- Analytics and reporting

## Installation

```
bench get-app https://github.com/your-org/mtk_property
bench install-app mtk_property
bench --site <site> execute mtk_property.api.setup.enable_mtk_property_for_site
```
